Case Studies

Real problems, real solutions

Three automation projects built for real-estate and finance operations — each replaced hours of manual work with reliable automated systems.

ENTRY / 01

Collection Management System

Problem

A real-estate firm tracked collections across multiple spreadsheets. Payment status was unclear, cheque due dates were missed, and monthly collection reports took 3–4 hours to prepare manually.

Solution

An automated collection tracking & reporting system that consolidates payment data, tracks cheque due dates, and generates management-ready PDF reports — with a Telegram bot sending daily reminders for upcoming cheques.

Tech Stack

  • Python for data processing
  • Excel as data source
  • PDF report generation
  • Telegram Bot API + cron scheduling

My Role

End-to-end: workflow design, automation scripts, reporting templates, and Telegram integration for daily alerts.

IMPACT / Reporting time: 3–4 hours → 5 minutes · Zero missed cheque dates since deployment
ENTRY / 02

Real Estate Booking & CRM System

Problem

A sales team had no centralized system for leads, unit availability, or reservation status. Agents worked from personal notes, follow-ups were inconsistent, and management had no pipeline visibility.

Solution

A complete reservation & CRM workflow: lead distribution, real-time unit inventory tracking, reservation dashboard, payment-offer generation, and automated commission calculations.

Tech Stack

  • Excel-based data layer
  • Python automation scripts
  • PDF payment-offer generation
  • HTML/CSS dashboards

My Role

Full ownership: requirements, workflow design, implementation, testing with real data, and training the sales team.

IMPACT / Real-time unit availability · Reservation processing −60% · Commission disputes eliminated
ENTRY / 03

WhatsApp Reporting & Follow-Up Automation

Problem

Management compiled daily reports manually from multiple sources; customer payment follow-up happened by phone — slow, inconsistent, and always delayed.

Solution

Automated workflows that pull data from Excel/ERP sources on schedule and deliver formatted reports and follow-up messages to management and customers via WhatsApp and Telegram — zero manual intervention.

Tech Stack

  • n8n workflow orchestration
  • WhatsApp Business API
  • Telegram Bot for internal alerts
  • Python + scheduled cron jobs

My Role

Architected the pipeline, configured n8n workflows, integrated messaging APIs, and set up fully autonomous daily operation.

IMPACT / 10+ hours/month saved · Reports auto-delivered 11:30 AM daily · Payment response rate improved
